# Baseline Comparison

This case study compares four advisory modes:

1. unguided LLM marketing advice;
2. generic marketing checklist;
3. beta LuxPilot flow;
4. governed LuxPilot architecture.

## Evaluation dimensions

- feasibility;
- actionability;
- explainability;
- reproducibility;
- user validation;
- auditability;
- capacity fit.

## Expected advantage

The governed architecture is not designed to produce more fluent advice. It is designed to produce recommendations that are constrained, traceable, replayable, and easier to validate.
