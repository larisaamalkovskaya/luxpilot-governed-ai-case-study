# Public Artifact Boundary

This repository demonstrates:

- architecture;
- reproducibility design;
- evaluation approach;
- synthetic fixtures;
- public schemas;
- prompt hygiene principles;
- audit/replay concepts.

It does not disclose:

- production rule-base;
- exact gate predicates;
- exact thresholds;
- production prompts;
- commercial templates;
- real user data;
- real golden traces.

The goal is to make the system inspectable as a research and portfolio artifact without disclosing commercial implementation details.
